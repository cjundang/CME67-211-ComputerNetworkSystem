# **2. คำอธิบายรายวิชา**

ศึกษาทฤษฎีและหลักการทำงานของระบบเครือข่ายคอมพิวเตอร์ ครอบคลุมแนวคิดด้านการสื่อสารข้อมูล สถาปัตยกรรมและแบบจำลองเครือข่าย โพรโทคอลการสื่อสาร อุปกรณ์เครือข่าย การออกแบบโทโปโลยี การรักษาความมั่นคงปลอดภัยระบบเครือข่าย และการทำงานของระบบอินเทอร์เน็ต ผู้เรียนจะได้ฝึกปฏิบัติการวิเคราะห์ ออกแบบ ติดตั้ง กำหนดค่า ทดสอบ และบำรุงรักษาระบบเครือข่ายคอมพิวเตอร์ภายในองค์กรและภายนอกองค์กร รวมถึงการปฏิบัติงานด้านระบบแม่ข่ายคอมพิวเตอร์อย่างเป็นระบบ

---

# **3. จุดประสงค์และผลการเรียนรู้ของรายวิชา**

## **3.1 จุดประสงค์รายวิชา**

1. เพื่อให้ผู้เรียนมีความรู้พื้นฐานเกี่ยวกับระบบเครือข่ายคอมพิวเตอร์และการสื่อสารข้อมูล
2. เพื่อพัฒนาความสามารถในการอธิบายแบบจำลองเครือข่ายและโพรโทคอลที่เกี่ยวข้อง
3. เพื่อเสริมสร้างทักษะในการติดตั้งและกำหนดค่าอุปกรณ์เครือข่าย
4. เพื่อให้ผู้เรียนสามารถวิเคราะห์และแก้ไขปัญหาเครือข่ายได้อย่างถูกต้อง
5. เพื่อส่งเสริมความสามารถในการออกแบบระบบเครือข่ายและสร้างโครงงานจริง
6. เพื่อปลูกฝังความรับผิดชอบและจรรยาบรรณทางวิชาชีพครูในการทำงานกลุ่มและโครงงาน

---

## **3.2 ผลการเรียนรู้ของรายวิชา (CLOs) – *ฉบับปรับให้สอดคล้อง PLO***

### **ด้านความรู้ (Knowledge) — PLO7, PLO8**

* **CLO1:** อธิบายองค์ประกอบ หลักการทำงาน และแบบจำลองของระบบเครือข่ายคอมพิวเตอร์ได้อย่างถูกต้อง (**PLO7**)
* **CLO2:** อธิบายการสื่อสารข้อมูล โครงสร้างเครือข่าย และแนวทางการรักษาความมั่นคงปลอดภัยของระบบได้อย่างเป็นระบบ (**PLO7**)
* **CLO3:** ประยุกต์ความรู้ด้านคอมพิวเตอร์และเทคโนโลยีเพื่อการจัดการเรียนรู้ในหัวข้อระบบเครือข่ายได้อย่างเหมาะสม (**PLO8**)

### **ด้านทักษะ (Skills) — PLO25**

* **CLO4:** ติดตั้งและกำหนดค่าอุปกรณ์เครือข่าย และทดสอบการทำงานได้อย่างถูกต้องตามหลักวิชาการ (**PLO25**)
* **CLO5:** ใช้เครื่องมือวิเคราะห์ปัญหาเครือข่ายและแก้ไขข้อขัดข้องได้อย่างมีประสิทธิภาพ (**PLO25**)
* **CLO6:** ออกแบบโครงสร้างระบบเครือข่ายคอมพิวเตอร์และพัฒนาโครงงานได้อย่างครบถ้วนและถูกต้อง (**PLO25**)

### **ด้านจริยธรรม (Ethics) — PLO26**

* **CLO7:** ประพฤติปฏิบัติตามจรรยาบรรณวิชาชีพครู ความรับผิดชอบ ความซื่อสัตย์ และการทำงานร่วมกับผู้อื่นในกิจกรรมและโครงงานเครือข่าย (**PLO26**)

# **4. วิธีการจัดการเรียนรู้ (Teaching and Learning Activities: TLA)**
